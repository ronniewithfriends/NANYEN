# MASCOT_GENERATION_PROMPTS.md

Use these prompts when generating expression variants for the five NANYEN mascots.

## Shared Constraints

Preserve:

- the original hand-drawn line wobble
- marker-like uneven color fill
- black outline
- childlike handmade feeling
- the same body shape, proportions, colors, clothing, and defining features
- white paper background

Change only:

- face / eyes / mouth / tiny emotional marks if needed

Avoid:

- polished vector style
- smooth AI mascot redesign
- changing pose or costume
- extra objects
- text
- watermark
- judgmental, angry, or mean expressions

## Brag / Smug Sparkle

Create a playful proud "doya" expression variant. The character should look happy and a little smug,
as if saying "today we can brag a bit." Friendly, not mean.

## Overspend / Blank Eyes Sweat

Create a gentle self-joking overspend expression variant. The character should look flustered,
blank-eyed, or lightly sweating, but still friendly. Do not make it scary, ashamed, or scolding.

## Empathy / Warm Smile

Create a warm empathy expression variant. The character should look relaxed, welcoming, and like it
is laughing together with the user. Keep it soft and companion-like.
